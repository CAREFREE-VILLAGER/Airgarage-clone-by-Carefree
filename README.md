# Project Name

AirGarage Parking Application

## Description

This project is a web application for managing parking spaces provided by AirGarage. Users can log in, find parking spots, and get support for various issues related to parking violations, rentals, and more.(UNDER PROGRESS)

## Table of Contents

- [Features](#features)
- [Installation](#installation)
- [Usage](#usage)
- [API Integration](#api-integration)


## Features

- **User Authentication**: Secure user login to access personalized features.
- **Find Parking**: Search for available parking spots based on location.
- **Customer Support Center**: Get assistance for parking violations, rental issues, and more.
- **Responsive Navbar**: Navigation bar that adapts to different pages.

## Installation

1. Clone the repository.
2. Install dependencies using `npm install`.
3. Run the application with `npm start`.

## Usage

1. Navigate to the login page and log in with your credentials.
2. Explore the application features, including finding parking spots and accessing the customer support center.

## API Integration

I have tried to use AirGarage API for fetching parking spot information. Ensure API keys and authentication are set up for seamless functionality.
